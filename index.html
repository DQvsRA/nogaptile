Features & Settings

There is some OPTIONS you can play with

Max Width & Height

You can specify maximum tile size:
- max_width_dimension (default 2)
- max_height_dimension (default 2)

Square Scale

That option allow proportional scale when resize happend. It's very fast cause all calculation cached and no need to be recalculated:
- squarescale (default false)

Gap Markup & Background

Set markup or background color for every gap that added to layout:
- gap_markup (default "")
- gap_backgound (default "#343434")

Gap Class

Specify class name for all gaps:
- gap_classname (default "gap")

Different Algorithm

There are a few algorithm for solve layout. You can choose static proportion, fit layout size insequense, move last tile to close gap.

Response

onResize ( cellwidth, columns )
onLayoutChanged ( tiles, columns )
